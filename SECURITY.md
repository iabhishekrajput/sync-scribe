# Security Policy

## Supported Versions

Currently, the `main` branch is the only actively supported version of Anekdote Auth.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| legacy  | :x:                |

## Reporting a Vulnerability

Security is a top priority for this Authorization Server. If you discover a vulnerability, **do not open a public issue.**

Please report any security issues privately by emailing **iabhishekrajput@gmail.com**. You should receive a response within 48 hours. 

When reporting, please provide:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact and, if possible, suggested mitigations.

Public disclosure is discouraged until a patch has been published.
